# golang dmidecode
> part of golang console

> 依赖 doggytty/goutils

> 支持用户: root/sudo user

### 命令:
> dmidecode -t bios

> dmidecode -t system

> dmidecode -t baseboard

> dmidecode -t chassis

> dmidecode -t processor

> dmidecode -t memory

> dmidecode -t cache

> dmidecode -t connector

> dmidecode -t slot
 
### 备注
> 测试环境linux/amd64(ubuntu 16.04 LTS)
